# BatmanGame 蝙蝠俠迷宮遊戲

### 遊戲背景:
蝙蝠俠-世界上最偉大的偵探，如今被邪惡的JOKER(小丑)困在了毒氣室，
小丑還聯合了其他罪犯HARLEY QUINN(小丑女)、RIDDLER(謎語人)、TWO FACE(雙面人)、
PENGUIN(企鵝人)，想要阻止蝙蝠俠脫逃毒氣室，快幫助蝙蝠俠逃離困境吧!!

### 遊戲進行方式:
起始位置B(Batman)表示玩家，使用鍵盤w、a、s、d(上、下、左、右)進行操作。
蝙蝠俠面對敵人時需比敵人的Level高才可以打敗敵人，
打敗敵人的同時蝙蝠俠會提升敵人的相對應等級。
迷宮中的各個英文字母代表敵人的縮寫H(Harley Quinn)、R(Riddler)、P(Penguin)、T(Two Face)、J(Joker)。
打敗JOKER後即可往JOKER下方的出口跑出迷宮過關。

蝠俠俠面對敵人時可以選擇4種攻擊方式:
1.Batman Kick
2.Batman Punch
3.Bat Mobile
4.Batman Batarang

##### 使用C++語言、物件導向、WindowsAPI
